The Legacy Proyect

**Principios de The Legacy Project by The Brother's of Core** 1. **Goal (Meta):** - Se deben implementar objetivos SMART para establecer metas claras y alcanzables. - Imagen mental: JL con un rifle francotirador en EE.UU., paseando al perro en Washington. - Relación con el número: El 1 se parece al rifle de francotirador. 2. **Quantification (Cuantificación):** - Es necesario crear un registro de información a aprender. - Imagen mental: Contando patos ("cuac cuac"). - Relación con el número: El 2 parece un pato. 3. **Association (Asociación):** - La información debe asociarse con recursos mentales conocidos para mejorar la retención. - Imagen mental: Asociarse con el diablo y ser malvado. - Relación con el número: El 3 es el diablo con dos puntos. 4. **Emotion (Emoción):** - Se debe aprovechar recuerdos intensos, sean positivos o negativos, para reforzar la memorización. - Imagen mental: Frente a Itachi Uchiha en el templo de los Uchiha, mientras se cumple la venganza. - Relación con el número: El 4 parece una silla al revés. 5. **Attention (Atención):** - Enfocar la atención en los elementos clave de la información a aprender. - Imagen mental: Una mujer embarazada, destacando que es ella quien necesita atención. - Relación con el número: El 5 se parece a una mujer embarazada. 6. **Order (Orden):** - La UI/UX debe ser cómoda, intuitiva y diseñada con enfoque Mobile First. - Imagen mental: Ordenando caracoles de Bob Esponja desordenados por todos lados. - Relación con el número: El 6 se parece a un caracol. 7. **Review (Revisión):** - Aplicar la Repetición Espaciada o el Método Leitner para reforzar el aprendizaje a lo largo del tiempo. - Imagen mental: Un abismo que observa al usuario, como en la idea de Nietzsche. - Relación con el número: El 7 parece un abismo. 8. **Imitation (Imitación):** - Aprender imitando a alguien a quien se quiera llegar a ser. - Imagen mental: Un muñeco de nieve que se parece a todos los demás muñecos de nieve. - Relación con el número: El 8 es el muñeco de nieve. 9. **Monitoring (Monitoreo):** - La app debe incluir alertas y recordatorios para el usuario, como si fueran alarmas para tomar medicina. - Imagen mental: Sentado viendo un monitor que muestra su progreso. - Relación con el número: El 9 parece una cabeza y una persona sentada. 10. **Feedback:** - Permitir pruebas constantes y recibir retroalimentación para mejorar el aprendizaje. - Imagen mental: Messi y CR7 dándose feedback mutuamente. - Relación con el número: El 10 es el número de Messi.
